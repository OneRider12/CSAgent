# Lumina Pulse Design System

Source: `raw stuff/DESIGN.md`

Lumina Pulse defines the visual system for the project.

## Key Points

- Brand direction: expressive, approachable, modern, and high contrast.
- Primary visual identity uses magenta-pink, sunflower yellow, and cool neutral surfaces.
- Typography uses Inter with clear headline, body, and label roles.
- Layout uses a fluid grid and 8px spacing rhythm.
- Components include primary and secondary buttons, outlined inputs, cards, chips, and tags.
