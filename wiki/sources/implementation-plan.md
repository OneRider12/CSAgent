# Implementation Plan

Source: `raw stuff/PLAN.md`

The implementation plan describes the LLM Class Wiki as a peer-to-peer knowledge hub for upload, sharing, AI processing, wiki browsing, graph exploration, chat, and Obsidian export.

## Phases

1. Foundation and Universal Uploads: repository setup, authentication, dashboard, upload, and storage.
2. AI Processing, Linting, and Human Review: OCR/parsing, strict LLM output, markdown linting, and review UI.
3. Automated Wiki and Link Creation: backlinks, wiki viewer, and graph view.
4. Chat and Export: RAG, cited AI chat, and Obsidian zip export.

## Quality Direction

The project emphasizes hybrid processing: AI drafts content, automated linting cleans it, and humans approve it before publishing.
